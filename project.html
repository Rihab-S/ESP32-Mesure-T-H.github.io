ESP32 Real-Time DHT & RSSI Dashboard

Developed an ESP32-based system to read temperature and humidity from a DHT11 sensor and measure WiFi RSSI. Data is displayed on an OLED screen and streamed to a real-time web dashboard. The dashboard includes live charts for temperature and humidity and allows CSV download of historical data.

Branchement des composants

  • DHT11
    • S (Signal) → GPIO4 (D4)
    • VCC (3.3V) → Rail d’alimentation positif (rouge)
    • GND (Ground) → Rail d’alimentation négatif (bleu)
  • OLED SSD1306 (I2C)
    • SDA → GPIO21 (D21)
    • SCL → GPIO22 (D22)
    • VCC (3.3V) → Rail d’alimentation positif (rouge)
    • GND (Ground) → Rail d’alimentation négatif (bleu)
  • Alimentation via la breadboard
    • ESP32 3.3V → Rail d’alimentation positif (rouge)
    • ESP32 GND → Rail d’alimentation négatif (bleu)
    • DHT11 et OLED prennent leur alimentation depuis ces rails

Résumé des GPIO utilisés

Fonction GPIO ESP32
DHT11 SignalGPIO4
OLED SDAGPIO21
OLED SCLGPIO22
Alimentation3.3V et GND via rails
